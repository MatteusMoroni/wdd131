Login Page

This project is an exercise in front-end fundamentals, demonstrating that a clean and responsive user interface can be achieved without relying on JavaScript. The focus was on writing clean, semantic HTML for accessibility and structure, paired with modern CSS for layout and styling. The page is fully responsive, adapting seamlessly to desktop and mobile devices using Flexbox and media queries. Subtle CSS transitions on input focus and button hover enhance the user experience, proving a strong command of core front-end technologies.
